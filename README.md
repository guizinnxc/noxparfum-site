# NOX Parfum Web — V1.0 Premium

## Como atualizar no GitHub
1. Abra o repositório no GitHub.
2. Clique em **Add file > Upload files**.
3. Envie todos os arquivos e pastas deste projeto.
4. Clique em **Commit changes**.
5. Aguarde 1 minuto e acesse https://noxparfum.com.br.

## Como trocar preço ou descrição
Edite o arquivo:
`data/perfumes.json`

Depois envie novamente para o GitHub.

## Como adicionar perfume novo
1. Coloque a imagem em `assets/images/products/`.
2. Adicione o produto no arquivo `data/perfumes.json`.
3. Crie uma página em `perfumes/` seguindo o modelo das páginas existentes.
